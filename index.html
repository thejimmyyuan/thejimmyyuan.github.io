Professional

Professionally, I've worked at multiple places throughout the summers of High School and College.

MasterCard WorldWide - Summer 2014

At MasterCard, I was assigned to re-create the original Publications website which was outdated. I implemented a string of new features including favoriting, sorting, filtering, and search using only JavaScript and XML. I also re-designed the layout of the site using User Experience principles. The new website is also a lot more automated than the old one, updating the site now only requires a file transfer, compared to tediously copying and pasting links in the past. The website is now on it's way to production.
